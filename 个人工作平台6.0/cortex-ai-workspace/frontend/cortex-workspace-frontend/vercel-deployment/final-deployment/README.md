# 🚀 立即部署到 Vercel

## 📱 3步完成部署

### 第1步: 访问 Vercel
- 打开: https://vercel.com
- 使用您的账户登录

### 第2步: 创建项目
- 点击 "New Project"
- 选择 "Upload" 或拖拽部署

### 第3步: 上传文件
- 拖拽整个 `dist` 文件夹到上传区域
- 等待部署完成 (30-60秒)
- 点击访问链接

## 🎯 目标地址
`https://cortex-ai-black.vercel.app`

## ⚙️ 配置说明
- **Framework**: 选择 "Vite" 或 "Other"
- **Root Directory**: `dist`
- **Build Command**: 留空 (已预构建)
- **Output Directory**: `dist`

## ✅ 部署验证
部署后访问您的 `.vercel.app` 链接，应该能看到：
- Cortex AI Workspace 登录页面
- 完整的界面设计
- 响应式布局

## ⚠️ 注意
由于后端在本地网络，API 功能需要额外配置。
